# Home Credit Default Risk Analysis

## Objective
Predict loan default risk and identify borrower characteristics associated with credit risk.

## Dataset
Home Credit Default Risk dataset from Kaggle (~300k loans).

## Key Insights
- High loan-to-income ratios significantly increase default risk.
- Borrowers with unstable employment show higher delinquency rates.
- Prior late payments strongly predict future default.

## Model
XGBoost

ROC AUC: 0.78

## Business Application
A risk-based approval strategy can reduce expected losses by 30%.
